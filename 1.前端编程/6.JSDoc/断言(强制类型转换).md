# 断言(强制类型转换)

## 概述

+ 通过在任何带括号的表达式前添加 `@type` 标记，可以将类型转换为其他类型

  ```js
  /**
   * @type {number | string}
   */
  var numberOrString = Math.random() < 0.5 ? "hello" : 100;
  var typeAssertedNumber = /** @type {number} */ (numberOrString);
  ```

+ 甚至可以像TypeScript一样强制转换为const:

  ```js
  let one = /** @type {const} */(1);
  ```
