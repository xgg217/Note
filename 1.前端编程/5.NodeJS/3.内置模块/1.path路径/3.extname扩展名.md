# extname 扩展名

## 概述

+ 返回 `path` 的扩展名

+ 即 `path` 的最后一部分中从最后一次出现的 `.`（句点）字符到字符串的结尾

+ 如果 `path` 的最后一部分中没有 `.`，或者除了 `pat`h 的基本名称（参见 `path.basename()` ）的第一个字符之外没有 `.` 个字符，则返回空字符串

## path.extname(path)

+ 参数

  + path `<string>`

+ 返回: `<string>`

  ```js
  path.extname('index.html'); // 返回: '.html'

  path.extname('index.coffee.md'); // 返回: '.md'

  path.extname('index.'); // 返回: '.'

  path.extname('index'); // 返回: ''

  path.extname('.index'); // 返回: ''
  ```
