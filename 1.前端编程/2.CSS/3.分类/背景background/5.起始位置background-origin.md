# background-origin

## 概述

+ 规定了指定背景图片background-image 属性的原点位置的背景相对区域

## 注意

+ 注意：当使用 background-attachment 为 fixed 时，该属性将被忽略不起作用

## 语法

+ 取值

  + `border-box` 从边框开始显示背景

    ![alt text](images/border-box.png)

  + `padding-box` 从内边距开始显示背景 默认值

    ![alt text](images/padding-box.png)

  + `content-box` 从内容开始显示背景

    ![alt text](images/content-box.png)

  ```css
  /* 关键字值 */
  background-origin: border-box;
  background-origin: padding-box;
  background-origin: content-box;

  /* 全局值 */
  background-origin: inherit;
  background-origin: initial;
  background-origin: revert;
  background-origin: revert-layer;
  background-origin: unset;
  ```
