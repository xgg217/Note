# background-size 背景图片大小

## 概述

+ background-size 设置背景图片大小
+ 图片可以保有其原有的尺寸，或者拉伸到新的尺寸，或者在保持其原有比例的同时缩放到元素的可用空间的尺寸

## 语法

+ `background-size: 宽 高;`

+ 默认值 `background-size: auto auto;`

+ 取值

  + 情况1 `定值 auto` 或 `auto 定值` 根据原图像的比例定制计算
  + 情况2 `cover` 覆盖，图片尽可能把当前元素覆盖，不留空白，可能出现裁剪
  + 情况3 `contain` 包含 图片尽可能的展示在元素中，可能出现留白

  ```css
  /* 关键字 */
  background-size: cover;
  background-size: contain;

  background-size: auto;
  /* 相等于 */
  background-size: auto, auto;

  /* 指定图片的宽度，图片的高度隐式的为 auto */
  background-size: 50%;
  background-size: 3.2em;
  background-size: 12px;

  /* 第一个值指定图片的宽度，第二个值指定图片的高度 */
  background-size: 50% auto;
  background-size: 3em 25%;
  background-size: auto 6px;
  background-size: auto auto;

  /* 全局属性 */
  background-size: inherit;
  background-size: initial;
  background-size: unset;
  ```

## 逗号分隔

+ 逗号分隔的多个值：设置多重背景

  ```css
  background-size: auto, auto     /* 不同于 background-size: auto auto */
  background-size: 50%, 25%, 25%
  background-size: 6px, auto, contain
  ```
