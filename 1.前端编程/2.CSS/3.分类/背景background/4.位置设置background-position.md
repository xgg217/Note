# background-position 位置设置

## 概述

+ 为每一个背景图片设置初始位置
+ 这个位置是相对于由 background-origin 定义的位置图层的

+ 复合属性

  + background-position-x
  + background-position-y

## 关于百分比

+ 剩余空间的百分比

+ 水平偏移 = （容器宽度 - 图像宽度） * （水平偏移百分比 * %）
+ 垂直偏移 = （容器高度 - 图像高度） * （垂直偏移百分比 * %）

## 语法

+ 一个值

  ```css
  /* 居中背景图片 */
  background-position: center;

  /* 关键字 top、left、bottom、right 中的一个,另一个维度被设置成 50% */
  background-position: top;
  background-position: bottom;
  background-position: left;
  background-position: right;

  /* 具体值 或 百分比： 指定相对于左边界的 x 坐标，y 坐标被设置成 50% */
  background-position: 10px;

  background-position: 10%;
  /* 相当于 */
  background-position: 10% 50%;
  ```

+ 两个值的语法： 水平偏移量 垂直偏移量

  + 即：一个定义 x 坐标，另一个定义 y 坐标

  ```css
  /* 情况1 关键字 top、left、bottom、right 中的一个 */
  background-position: left top;

  /* 情况2 具体值 或 百分比 */
  background-position: 25% 75%;

  /* <length> values */
  background-position: 0 0;
  background-position: 1cm 2cm;
  background-position: 10ch 8em;
  ```

+ 三个值与四个值： 水平方向 偏移量 垂直方向 偏移量

  + 偏移量缺失，默认值为0

  ```css
  background-position: left 10px top 15px; /* 10px 15px */
  background-position: left      top     ; /*  0px  0px */
  background-position:      10px     15px; /* 10px 15px */
  background-position: left          15px; /*  0px  15px */
  background-position:      10px top     ; /* 10px  0px */
  background-position: left      top 15px; /*  0px 15px */
  background-position: left 10px top     ; /* 10px  0px */
  ```

+ 全局值

  ```css
  /* Global values */
  background-position: inherit;
  background-position: initial;
  background-position: revert;
  background-position: unset;
  ```

+ 示例

  ```css
  background-position: top;
  background-position: left;
  background-position: center;
  background-position: 25% 75%;
  background-position: bottom 50px right 100px;
  background-position: right 35% bottom 45%;
  ```

## 注意点

+ 如果值是 20px left 两个水平偏移量是无效的

  ```css
  /* 无效属性 */
  background-position: 20px left;
  ```
