# background 背景图

## 概述

+ background 是一种 CSS 简写属性，用于一次性集中定义各种背景属性，包括 color, image, origin 与 size, repeat 方式等等

## background 语法

+ 此属性是一个简写属性，可以在一次声明中定义一个或多个属性：background-clip、background-color、background-image、background-origin、background-position、background-repeat、background-size 和 background-attachment

  ```css
  /* 使用 <background-color> */
  background: green;

  /* 使用 <bg-image> 和 <repeat-style> */
  background: url("test.jpg") repeat-y;

  /* 使用 <box> 和 <background-color> */
  background: border-box red;

  /* 将背景设为一张居中放大的图片 */
  background: no-repeat center/80% url("../img/image.png");


  background: content-box radial-gradient(crimson, skyblue);

  background: no-repeat url("/shared-assets/images/examples/lizard.png");

  background: left 5% / 15% 60% repeat-x url("/shared-assets/images/examples/star.png");
  ```

## 多个背景层

+ 属性被指定多个背景层时，使用逗号分隔每个背景层

+ 每一层的语法如下：

  + 在每一层中，下列的值可以出现 0 次或 1 次：

    + `<attachment>`
    + `<bg-image>`
    + `<position>`
    + `<bg-size>`
    + `<repeat-style>`

+ `<bg-size>` 只能紧接着 `<position>` 出现，以"/"分割，如： "center/80%".

+ `<box>` 可能出现 0 次、1 次或 2 次。如果出现 1 次，它同时设定 background-origin 和 background-clip。如果出现 2 次，第一次的出现设置 background-origin，第二次的出现设置 background-clip

+ `<background-color>` 只能在 background 的最后一个属性上定义，因为整个元素只有一种背景颜色

  ```css
  background: center / contain no-repeat url("/shared-assets/images/examples/firefox-logo.svg"), #eee 35% url("/shared-assets/images/examples/lizard.png");
  ```

## 背景图属性

+ background-clip 裁剪背景
+ background-color
+ `background-image`：背景图路径（可写可不写 `""`，如果路径中包含特殊符号，需要写 `""`）

+ background-origin

+ `background-position`：设置背景图的位置

+ `background-repeat`：背景在 `x` 、 `y` 轴进行重复

+ `background-size`：尺寸

+ `background-attachment`：控制被进图是否固定（类似固定定位）
