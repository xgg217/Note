# 图片路径 background-image

## 概述

+ 为一个元素设置一个或者多个背景图像

+ 在绘制时，图像以 z 方向堆叠的方式进行。先指定的图像会在之后指定的图像上面绘制
+ 因此指定的第一个图像“最接近用户”

## 语法

+ `none` 是一个表示无背景图的关键字

+ `url` 用来标记将要显示的图片。支持多背景设置，背景之间以逗号隔开

  ```css
  background-image: url("/shared-assets/images/examples/lizard.png");

  background-image: url("/shared-assets/images/examples/lizard.png"), url("/shared-assets/images/examples/star.png");
  ```

  ```css
  background-image:
    linear-gradient(to bottom, rgba(255, 255, 0, 0.5), rgba(0, 0, 255, 0.5)),
    url("cat-front.png");
  ```
