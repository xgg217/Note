# 过渡动画 transition

## 本质

+ 从一个状态过渡到一个状态 transition:

  + 如何切换状态
  + 哪（些）个属性参与了变化？（transiton-property）
  + 用多长时间完成过渡？(transition-duration)
  + 过渡的状态，是快是慢，还是先快后慢, (transition-timing-function)
  + 提前开始过渡，延后开始过渡(transition-delay)

## 如何切换状态

+ 可以利用伪类 :hover 也可通过 js 操控（修改类名，修改属性）

## 概述

+ transition CSS 属性是 `transition-property` 、 `transition-duration` 、 `transition-timing-function` 和 `transition-delay` 的一个简写属性。

  ```css
  /* 过渡属性 */
  transition-property

  /* 过渡时长 */
  transition-duration

  /* 过渡函数 */
  transition-timing-function

  /* 延迟时间 */
  transition-delay
  ```

## 语法

+ 语法

  ```css
  transition: margin-right 4s;

  transition: margin-right 4s 1s;

  transition: margin-right 4s ease-in-out;

  transition: margin-right 4s ease-in-out 1s;

  transition:
    margin-right 4s,
    color 1s;

  transition: all 0.5s ease-out;

  transition: inherit;
  transition: initial;
  transition: unset;
  ```

  ```css
  .box {
    transition-property: background-color, widht;
    transition-duration: 1s, 1.5s;
  }

  .box:hover {
    width: 300px;
    background-color: #0f0;
  }
  ```
