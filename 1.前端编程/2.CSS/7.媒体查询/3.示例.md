# 示例

## 示例1：响应式网格布局

+ 响应式网格布局

  ```html
  <!DOCTYPE html>
  <html>
  <head>
    <style>
      .grid-container {
        display: grid;
        grid-template-columns: repeat(12, 1fr);
        gap: 10px;
        padding: 10px;
      }

      .grid-item {
        background-color: #4CAF50;
        padding: 20px;
        text-align: center;
        color: white;
        grid-column: span 12;
      }

      /* 平板设备 */
      @media screen and (min-width: 768px) {
        .grid-item {
          grid-column: span 6;
        }

        .grid-item.featured {
          grid-column: span 12;
        }
      }

      /* 桌面设备 */
      @media screen and (min-width: 1024px) {
        .grid-item {
          grid-column: span 4;
        }

        .grid-item.featured {
          grid-column: span 8;
        }

        .grid-item.highlight {
          grid-column: span 6;
        }
      }
    </style>
  </head>
  <body>
    <div class="grid-container">
      <div class="grid-item featured">特色内容</div>
      <div class="grid-item">内容1</div>
      <div class="grid-item">内容2</div>
      <div class="grid-item highlight">高亮内容</div>
      <div class="grid-item">内容3</div>
      <div class="grid-item">内容4</div>
    </div>
  </body>
  </html>

  ```

## 示例2：响应式导航菜单

+ 响应式导航菜单

  ```html
  <!DOCTYPE html>
  <html>
  <head>
  <style>
    nav {
      background-color: #333;
      overflow: hidden;
    }

    nav ul {
      list-style-type: none;
      margin: 0;
      padding: 0;
    }

    nav li {
      float: left;
    }

    nav a {
      display: block;
      color: white;
      text-align: center;
      padding: 14px 20px;
      text-decoration: none;
    }

    nav a:hover {
      background-color: #111;
    }

    /* 移动设备样式 */
    @media screen and (max-width: 768px) {
      nav li {
        float: none;
      }

      .menu-toggle {
        display: block;
        color: white;
        text-align: center;
        padding: 14px 20px;
        text-decoration: none;
        cursor: pointer;
      }

      nav ul {
        display: none;
      }

      nav ul.active {
        display: block;
      }
    }

    /* 桌面设备样式 */
    @media screen and (min-width: 769px) {
      .menu-toggle {
        display: none;
      }

      nav ul {
        display: block !important;
      }
    }
  </style>
  </head>
  <body>
    <nav>
      <a class="menu-toggle">菜单</a>
      <ul>
        <li><a href="#home">首页</a></li>
        <li><a href="#about">关于</a></li>
        <li><a href="#services">服务</a></li>
        <li><a href="#contact">联系</a></li>
      </ul>
    </nav>
  </body>
  </html>

  ```

## 示例3：响应式图片

+ 响应式图片

  ```html
  <!DOCTYPE html>
  <html>
  <head>
  <style>
    .responsive-image {
      width: 100%;
      height: auto;
    }

    .image-container {
      max-width: 1200px;
      margin: 0 auto;
      padding: 20px;
    }

    /* 小屏设备 */
    @media screen and (max-width: 480px) {
      .image-container {
        padding: 10px;
      }

      .responsive-image {
        border-radius: 5px;
      }
    }

    /* 中等屏设备 */
    @media screen and (min-width: 481px) and (max-width: 768px) {
      .image-container {
        padding: 15px;
      }

      .responsive-image {
        border-radius: 10px;
        box-shadow: 0 4px 8px rgba(0,0,0,0.1);
      }
    }

    /* 大屏设备 */
    @media screen and (min-width: 769px) {
      .image-container {
        padding: 30px;
      }

      .responsive-image {
        border-radius: 15px;
        box-shadow: 0 8px 16px rgba(0,0,0,0.2);
      }
    }
  </style>
  </head>
  <body>
    <div class="image-container">
      <img src="https://via.placeholder.com/800x400" alt="响应式图片" class="responsive-image">
    </div>
  </body>
  </html>

  ```
