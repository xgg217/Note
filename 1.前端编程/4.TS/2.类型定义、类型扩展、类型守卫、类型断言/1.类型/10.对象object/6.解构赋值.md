# 解构赋值

## 概述

+ 解构赋值用于直接从对象中提取属性

  ```js
  const {id, name, price} = product;
  ```

## 解构赋值的类型写法

+ 解构赋值的类型写法，跟为对象声明类型是一样的

  ```js
  const {id, name, price}:{
    id: string;
    name: string;
    price: number
  } = product;
  ```

+ 注意，目前没法为解构变量指定类型，因为对象解构里面的冒号，JavaScript 指定了其他用途

  ```js
  let { x: foo, y: bar } = obj;

  // 等同于
  let foo = obj.x;
  let bar = obj.y;

  // 如果要为x和y指定类型，不得不写成下面这样
  let { x: foo, y: bar }
  : { x: string; y: number } = obj;
  ```
