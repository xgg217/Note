# number

## 数值类型 number

+ 不分是整数还是小数

  ```js
  let flag:number = 123;
  flag = 123;

  flag = false; // 错误
  ```

  ```js
  var age:number = 18
  var stature:number = 178.5
  console.log(age); // 18
  console.log(stature); // 178.5
  ```

+ 范围

  ```js
  {-Infinity, ..., -5, 0, 1, 3.14, 42, ..., Infinity}
  ```
