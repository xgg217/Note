# 分布式条件类型

## 分布式条件类型

+ 在条件类型（ `T extends U ? X : Y`）中，如果被检查的类型是一个“裸”类型参数，则被称为分布式条件类型

+ 裸类型

  + 没有被 数组（`T[]`）、元组（`[T]`）、Promise（`Promise<T>`）等包装过的类型

+ 参数

  + 下面的工具类型中，泛型表达式中的 `T` 就是类型参数

   ```js
    type IsString<T> = T extends string ? true : false
    ```
