# keyof

## VS Code 技巧

+ 在 VS Code 中悬浮鼠标只能看到 keyof User，看不到其中的实际值，你可以这么做

  ```js
  type User = {
    id: number
    name: string
    age: number
  }

  type UserKeys = keyof User & {} // "id" | "name" | "age"
  ```

## 类型别名 type

+ keyof 操作符它可以将对象中的所有键转换为对应字面量类型，然后再组合成联合类型

  ```js
  type User = {
    id: number
    name: string
    age: number
  }

  type UserKeys = keyof User; // 'id' | 'name' | 'age'
  ```

## 接口 interface

  ```js
  // 接口
  interface T {
    0: boolean;
    a: string;
    b(): void;
  }

  type KeyT = keyof T; // 0 | 'a' | 'b'
  ```

## 对象

+ 直接从一个对象上，获取这个对象键的所有联合类型

  ```js
  const user = {
    id: 1,
    name: 'hayes',
    age: 19
  };

  keyof user // "id" | "name" | "age"

  type UserKeys = keyof typeof user // "id" | "name" | "age"
  ```

## 数组 与 元组 使用 keyof

+ keyof 运算符用于数组或元组类型，会返回数组的所有键名，包括数字键名和继承的键名

  ```js
  type Result = keyof ['a', 'b', 'c'];

  // 返回 number | "0" | "1" | "2"
  // | "length" | "pop" | "push" | ···
  ```

## 方括号运算符结合

+ 方括号运算符结合

  ```js
  type Person = {
    age: number;
    name: string;
    sex: boolean;
  };

  // number|string|boolean
  type A = Person[keyof Person];
  ```

## keyof 运算符的用途

+ 取出对象的某个指定属性的值

  ```js
  function prop<Obj, K extends keyof Obj>(obj:Obj, key:K):Obj[K] {
    return obj[key];
  }
  ```

+ 属性映射:即将一个类型的所有属性逐一映射成其他值

  ```js
  type NewProps<Obj> = {
    [Prop in keyof Obj]: boolean;
  };

  // 用法
  type MyObj = { foo: number; };

  // 等于 { foo: boolean; }
  type NewObj = NewProps<MyObj>;
  ```
