# ||= 逻辑或赋值

## 语法

+ 逻辑或赋值（x ||= y）运算仅在 x 为假值时为其赋值

+ 语法

  ```js
  x ||= y

  // 等于
  x || (x = y);
  ```

+ 如果 x 为真值，则根本不会对 y 求值

  ```js
  const x = 1;
  x ||= console.log("y 进行了求值");
  // 什么都不会输出
  ```

## 注意

+ 如果返回的是空字符串（是假值），则必须使用 `||=`
+ 如果接口返回 `null` 或 `undefined` ，则应该使用 `??=`

## 示例

+ 示例

  ```js
  const a = { duration: 50, title: '' };

  a.duration ||= 10;
  console.log(a.duration); // 50

  a.title ||= 'title is empty.';
  console.log(a.title); // title is empty.

  a.b ||= 't.';
  console.log(a.b); // t.
  ```
