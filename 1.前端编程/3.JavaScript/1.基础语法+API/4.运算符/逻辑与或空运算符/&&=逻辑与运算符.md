# &&= 逻辑与运算符

## 概述

+ x &&= y）运算仅在 x 为真值时为其赋值

+ 语法

  ```js
  x &&= y

  // 等于
  x && (x = y);
  ```

+ 注意

  + 如果 x 不为真值，则根本不会对 y 求值

    ```js
    const x = 0;
    x &&= console.log("y 进行了求值");
    // 什么都不会输出
    ```

## 示例

+ 示例1

  ```js
  let a = 1;
  let b = 0;

  a &&= 2;
  console.log(a); // 2
  // expected output: 2

  b &&= 2;
  console.log(b); // 0
  ```
