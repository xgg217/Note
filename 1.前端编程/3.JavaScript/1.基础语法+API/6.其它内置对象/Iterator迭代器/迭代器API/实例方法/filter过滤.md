# Iterator.prototype.filter

## 概述

+ Iterator 实例的 filter() 方法返回一个新的迭代器辅助方法，该迭代器辅助方法只生成迭代器中能令提供的回调函数返回 true 的元素

+ 迭代器辅助方法相对于数组方法的主要优势在于它们能够处理无限迭代器
+ 对于无限迭代器，filter() 允许你只迭代满足给定条件的元素

## 语法

+ `filter(callbackFn)`

+ 参数

  + callbackFn 为迭代器中的每个元素执行的函数。它应该返回一个真值来使迭代器辅助方法产生对应元素，否则返回一个假值。该函数被调用时将传入以下参数：

    + element 当前正在处理的元素
    + index 当前正在处理的元素的索引

+ 返回值

  + 一个新的迭代器辅助方法
  + 每当迭代器辅助方法的 next() 方法被调用时，它都会返回一个能令回调函数返回 true 的元素
  + 当底层迭代器完成时，迭代器辅助方法也完成（next() 方法产生 { value: undefined, done: true }）
