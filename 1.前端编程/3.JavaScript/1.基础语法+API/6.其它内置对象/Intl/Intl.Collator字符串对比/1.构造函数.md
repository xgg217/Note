# 基础

## 解决的问题

+ 能够智能地根据不同语言和地区的规则进行字符串比较，让您的列表展示更加专业和用户友好

+ 例如，当您使用默认的 sort() 方法对包含中文、英文和数字的列表进行排序时，可能会出现意想不到的混乱结果

  + 数字排序错误：10排在2前面
  + 中文排序混乱：笔画顺序不准确
  + 特殊字符处理不当：标点符号影响排序逻辑
  + 大小写敏感问题：影响用户体验

## new Intl.Collator

+ `Intl.Collator()` 可以在使用或不适用 new 的情况下调用

  + 两者均能创建一个新的 Intl.Collator 实例

+ 语法

  ```js
  new Intl.Collator()
  new Intl.Collator(locales)
  new Intl.Collator(locales, options)

  Intl.Collator()
  Intl.Collator(locales)
  Intl.Collator(locales, options)
  ```
