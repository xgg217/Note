# 构造函数

## new Intl.Collator

+ `Intl.Collator()` 可以在使用或不适用 new 的情况下调用

  + 两者均能创建一个新的 Intl.Collator 实例

+ 语法

  ```js
  new Intl.Collator()
  new Intl.Collator(locales)
  new Intl.Collator(locales, options)

  Intl.Collator()
  Intl.Collator(locales)
  Intl.Collator(locales, options)
  ```
