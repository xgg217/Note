# 基础

## 概述

+ `Intl.NumberFormat` 是对语言敏感的格式化数字类的构造器类

  + [MDN 参考](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/NumberFormat)

  ```js
  const number = 123456.789;

  new Intl.NumberFormat("zh-CN").format(number);
  // 123,456.789

  new Intl.NumberFormat("de-DE", { style: "currency", currency: "EUR" }).format(number);
  // 123.456,79 €

  new Intl.NumberFormat("ja-JP", { style: "currency", currency: "JPY" }).format(number);
  // ￥123,457

  new Intl.NumberFormat("en-IN", { maximumSignificantDigits: 3 }).format(number);
  // 1,23,000
  ```
