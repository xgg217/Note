# slice

## 语法

+ `arr.slice(start_index, upto_index);`

+ `slice` 方法用于提取原数组的一部分，返回一个新数组

  + 它的第一个参数为起始位置（从0开始）
  + 第二个参数为终止位置（但该位置的元素本身不包括在内）

    + 如果省略第二个参数，则一直返回到原数组的最后一个成员

  ```js
  // 用法
  var a = ['a', 'b', 'c'];

  a.slice(0) // ["a", "b", "c"]
  a.slice(1) // ["b", "c"]
  a.slice(1, 2) // ["b"]
  a.slice(2, 6) // ["c"]
  ```

## 注意

+ 原数组不变

+ 没有参数:返回一个原数组的拷贝

  ```js
  var a = ['a', 'b', 'c'];
  a.slice()  // ["a", "b", "c"]
  ```

+ 参数是负数,则表示倒数计算的位置

  ```js
  var a = ['a', 'b', 'c'];
  a.slice(-2) // ["b", "c"]
  a.slice(-2, -1) // ["b"]
  ```

+ 如果参数值大于数组成员的个数，或者第二个参数小于第一个参数，则返回空数组

  ```js
  var a = ['a', 'b', 'c'];
  a.slice(4) // []
  a.slice(2, 1) // []
  ```
