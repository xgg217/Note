# 基础

## 事件处理程序

+ **某个 DOM** 发生了**某件事**之后，我们需要做的**某些处理**

  + 某个DOM：监听谁
  + 某件事（事件类型）：它发生了什么
  + 某些处理（处理函数）：我们要做什么

  ```js
  // 为 DOM 注册点击事件
  // 当被点击时，自动运行事件处理函数
  dom.onclick = function() {
    console.log('DOM 被点击了')
  }
  ```

## 事件注册

+ 将一个事件处理程序,挂载到某个事件上

## 事件流

+ 当某个事件发生的时候，哪些元素会监听到该事件发生。这些元素发生该事件的顺序
