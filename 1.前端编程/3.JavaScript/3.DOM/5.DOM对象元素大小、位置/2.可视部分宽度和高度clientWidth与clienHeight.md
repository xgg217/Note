# clientWidth 与 clienHeight

## 概述

+ 元素的可视部分宽度和高度( padding + content )，但不包括滚动条宽度（scrollbar）：

+ *只读属性*

  ```html
  <style>
    #example {
      width: 300px;
      height: 200px;
      border: 25px solid #E8C48F;
      padding: 20px;
      overflow: auto;
    }
  </style>

  <div id="example">
    ...Text...
  </div>

  <script>
    example.clientWidth // 20 + 284 + 20 = 324
    example.clienHeight // 20 + 200 + 20 = 240
  </script>
  ```

  ![alt text](images/clientWidth与clienHeight.png)

+ 如果没有 padding，那么 `clientWidth` / `clientHeight` 代表的就是内容区域，即 `border` 和 scrollbar（如果有）内的区域
