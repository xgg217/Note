# 基本使用

## 组件中使用

+ 使用方式1 `$t`

  ```html
  <div>{{ $t("messages.username") }}</div>
  ```

+ 使用方式2 `v-t`

  ```html
  <div v-t="'messages.username'"></div>
  ```

+ 使用方式3 `t`

  ```js
  import { useI18n } from "vue-i18n";

  const { t } = useI18n();

  t("message.title", { name: "三" });

  <div>{{ t("message.title", { name: "张三" }) }}</div>
  ```

+ 使用方式4（JS文件）

  ```js
  import i18n from "@/i18n/index";

  const t = i18n.global.t;

  export const name = t("message.title", { name: "三" });
  ```

## 支持变量

+ 支持变量

  ```js
  // 在你的语言文件中，例如：en.js
  export default {
    message: {
      hello: 'Hello, {name}!'
    }
  }
  ```

+ 页面中使用变量

  ```js
  <template>
    <p>{{ $t("message.hello", { name: '张三' }) }}</p>
  </template>
  ```

## 语言切换

+ 语言切换

  ```js
  import { useI18n } from "vue-i18n";
  const { locale } = useI18n();

  const changeLang = (lang: 'zh' | 'en') => {
    locale.value = lang;

    localStorage.setItem("lang", lang); // 本地缓存
  }
  ```
