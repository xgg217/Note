# CSS中的国际化

## 概述

+ 文案内容导致的CSS变化(中文、英文)

  + 使用字体族
  + 宽度或高度由内容决定
  + 设置最小或最大宽度
  + 使用 `text-overflow` 属性
  + 调整行高
  + CSS Grid

+ `[lang=locale]`

  + [MDN](https://developer.mozilla.org/zh-CN/docs/Web/HTML/Reference/Global_attributes/lang)

+ `:lang=locale`

  + [MDN](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Reference/Selectors/:lang)

+ Writing mode 块和行级内容的流动方向

  + horizontal-tb
  + vertical-rl
  + vertical-lr
  + sideways-rl
  + sideways-lr

  + text-orientation 行中字符的方向（仅影响纵向模式）

    + mixed
    + upright
    + sideways

+ text-combine-upright 竖行横书组合

+ direction 设置文本、表格列和水平溢出的方向

  + ltr
  + rtl

+ 逻辑属性

+ OpenType 字体特性指南
