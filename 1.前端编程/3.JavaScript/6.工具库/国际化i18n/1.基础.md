# 基础

## 术语

+ 国际化 internationalization 简称 i18n

  + 国际化是一种产品、应用程序或文档的设计和开发、可以方便为不同文化、地区或语言的目标受众进行本地化

+ 本地化 localization  简称 l10n
+ 全球化 globalization 简称 g11n

  + 国际化 + 本地化 = 全球化
