# xor

## 概述

+ xor 的作用是传递两个数组进去，第一个数组有第二个数组没有，或第二个数组有第一个数组没有的元素都会返回

## API

+ `function xor<T>(arr1: T[], arr2: T[]): T[];`

+ 参数

  + arr1 (T[]): 第一个数组。
  + arr2 (T[]): 第二个数组。

+ 返回值

  + (T[]): 包含在 arr1 或 arr2 中存在但不同时存在于两者中的元素的数组

  ```js
  // 返回 [1, 2, 5, 6]
  xor([1, 2, 3, 4], [3, 4, 5, 6]);

  // 返回 ['a', 'c']
  xor(['a', 'b'], ['b', 'c']);
  ```
