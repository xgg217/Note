# 函数组合

## flow

+ `_.flow([funcs])` 创建一个函数。 返回的结果是调用提供函数的结果，this 会绑定到创建函数。 每一个连续调用，传入的参数都是前一个函数返回的结果


## flowRight

+ `_.flowRight([funcs])` 这个方法类似_.flow，除了它调用函数的顺序是从右往左的

  ```js
  function square(n) {
    return n * n;
  }

  var addSquare = _.flowRight([square, _.add]);
  addSquare(1, 2);
  // => 9
  ```
