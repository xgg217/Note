# 选择 + 省略 + 可选 + 必填

## .pick() 选择

+ 选择某些键

  ```js
  const Recipe = z.object({
    title: z.string(),
    description: z.string().optional(),
    ingredients: z.array(z.string()),
  });

  const JustTheTitle = Recipe.pick({ title: true });
  // { title: string }
  ```

## .omit() 剔除

+ 要省略某些键：

  ```js
  const Recipe = z.object({
    title: z.string(),
    description: z.string().optional(),
    ingredients: z.array(z.string()),
  });

  const JustTheTitle = Recipe.omit({ id: true });
  // { description: string | undefined, ingredients: string[] }
  ```

## 指定属性可选 .optional()

+ 默认情况下，所有属性都是必需的
+ 要使某些属性可选

  ```js
  const Dog = z.object({
    name: z.string(),
    age: z.number().optional(),
  });

  Dog.parse({ name: "Yeller" }); // ✅
  ```

## 对象属性 可选批量操作 .optional()

+ 用于将部分或全部属性设为可选属性

  ```js
  const Recipe = z.object({
    title: z.string(),
    description: z.string().optional(),
    ingredients: z.array(z.string()),
  });

  // 所有字段可选
  const PartialRecipe = Recipe.partial();
  {
    title?: string | undefined;
    description?: string | undefined;
    ingredients?: string[] | undefined
  }
  ```

+ 指定属性可选

  ```js
  const RecipeOptionalIngredients = Recipe.partial({
    ingredients: true,
  });
  {
    title: string;
    description?: string | undefined;
    ingredients?: string[] | undefined
  }
  ```

## 必填 required()

+ 所有必填

  ```js
  const Recipe = z.object({
    title: z.string(),
    description: z.string().optional(),
    ingredients: z.array(z.string()),
  });

  const RequiredRecipe = Recipe.required();
  // { title: string; description: string; ingredients: string[] }
  ```

+ 要使某些属性成为必需属性

  ```js
  const Recipe = z.object({
    title: z.string(),
    description: z.string().optional(),
    ingredients: z.array(z.string()),
  });

  const RecipeRequiredDescription = Recipe.required({description: true});
  // { title: string; description: string; ingredients: string[] }
  ```
