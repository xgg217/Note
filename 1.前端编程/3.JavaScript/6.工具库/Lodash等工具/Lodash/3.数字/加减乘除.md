# 加减乘除

## add 加

+ `_.add(augend, addend)`

  ```js
  _.add(6, 4);
  // => 10
  ```

## subtract 减

+ `_.subtract(minuend, subtrahend)`

  ```js
  _.subtract(6, 4);
  // => 2
  ```

## multiply 乘

+ `_.multiply(multiplier, multiplicand)`

  ```js
  _.multiply(6, 4);

  // => 24
  ```

## divide 除

+ `_.divide(dividend, divisor)`

  ```js
  _.divide(6, 4);

  // => 1.5
  ```
