# 交错动画方向

## 概述

+ `anime.stagger(value, {reversed: true,})`

+ 更改交错动画的顺序

  + normal（默认），正常交错方向，从第一个元素到最后一个元素
  + reverse，倒退交错方向，从最后一个元素到第一个元素

  ```js
  animate('.staggering-direction-demo .el',{
    translateX: 270,
    delay: anime.stagger(100, {reversed: true,})
  });
  ```
