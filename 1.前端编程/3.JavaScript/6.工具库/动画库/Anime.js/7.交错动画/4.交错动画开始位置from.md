# 交错动画开始位置

## 概述

+ `anime.stagger(value, {from: startingPosition})` 从哪个位置依次延迟

  + from

    + first(默认)，从第一个元素开始效果
    + laster，从最后一个元素开始效果
    + center，从中间开始效果
    + index，从指定索引开始效果

  ```js
  animate('.staggering-from-demo .el',{
    translateX: 270,
    delay: anime.stagger(100, {from: 'center'})   //从中间开始执行效果，然后每个元素增加100毫秒
  });
  ```
