# 删除目标remove

## 概述

+ 从正在运行的动画或时间轴中删除目标。targets参数接受与targets 属性相同的值

  ```js
  animate('.remove-demo .el',{
    translateX: 270,
    alternate: true,
    loop: true,
    ease: 'inOutQuad'
  });

  document.querySelector('.remove-el-button').addEventListener('click', function() {
    utils.remove('.remove-demo .line:nth-child(2) .el');
  });
  ```
