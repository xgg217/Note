# 给每个属性单独设置

## 概述

+ code

  ```js
  import { animate } from 'animejs';

  animate('div',{
    translateX: {
      to: 250,//通过value来设置值
      duration: 800 //表示延长动画效果的时间
    },
    rotate: {
      to: 360,
      duration: 1800,
      ease: 'inOutSine'
    },
    scale: {
      to: 2,
      duration: 1600,
      delay: 800,
      ease: 'inOutQuart'
    },
    delay: 250 // 除‘scale’外的所有属性都继承250ms延迟
  });
  ```
