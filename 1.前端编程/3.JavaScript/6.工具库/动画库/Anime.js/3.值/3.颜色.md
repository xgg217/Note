# 颜色

## 概述

+ anime.js 接受并转换Haxadecimal(十六进制)，RGB，RGBA，HSL和HSLA颜色值
+ 注：不支持颜色名称，比如red

  ```
  接受             例
  十六进制         '#FFF'或'#FFFFFF'
  RGB的           'rgb(255, 255, 255)'
  RGBA 格式       'rgba(255, 255, 255, .2)'
  HSL            'hsl(0, 100%, 100%)'
  HSLA           'hsla(0, 100%, 100%, .2)'
  ```

  ```js
  var colorsExamples = createTimeline({
    loopDelay: 1000,
    ease: 'inOutQuad',
    alternate: true,
    loop: true
  })
  .add('.color-hex',{ background: '#FFF' }, 0)
  .add('.color-rgb',{ background: 'rgb(255,255,255)' }, 0)
  .add('.color-hsl',{ background: 'hsl(0, 100%, 100%)' }, 0)
  .add('.color-rgba',{ background: 'rgba(255,255,255, .2)' }, 0)
  .add('.color-hsla',{ background: 'hsla(0, 100%, 100%, .2)' }, 0)
  .add('.colors-demo .el',{ translateX: 270 }, 0);
  ```
