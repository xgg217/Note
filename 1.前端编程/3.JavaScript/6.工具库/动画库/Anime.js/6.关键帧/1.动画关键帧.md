# 动画关键帧

## 概述

+ 动画关键帧是使用keyframes属性中的数组定义的
+ 如果关键帧内没有指定duration（持续时间），则每个关键帧的持续时间将等于动画总持续时间除以关键帧数

  ```js
  animate('.item',{
    keyframes: [
      { translateY: -40, duration: 1000 },
      { translateX: 250, duration: 1000 },
      { translateY: 40, duration: 1000 },
      { translateX: 0, duration: 500 },
      { translateY: 0, duration: 500 }
    ],
    duration: 4000,
    ease: 'outElastic(1, .8)',
    loop: true
  })
  ```
