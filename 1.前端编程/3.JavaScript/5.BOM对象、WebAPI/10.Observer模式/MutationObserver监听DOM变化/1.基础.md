# 基础

## 概述

+ MutationObserver 是 JavaScript 中用于监听DOM树变化的接口
+ 它可以在 DOM 节点被添加、删除或修改时触发回调函数
+ 与旧的 Mutation Events 相比，MutationObserver更加高效和灵活

  ![alt text](images/MutationObserver.png)

## 使用场景

+ 动态加载内容时，监听新元素的插入
+ 监听元素的属性变化，如class、style等
+ 监听子节点的变化，如添加或删除子元素
