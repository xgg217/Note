# 使用

## 示例

+ 主线程代码

  ```js
  const worker = new Worker('./worker.js');

  // 消息监听
  worker.onmessage = e => {

  }

  // 错误处理
  worker.onerror = err => {

  }

  // 发送计算任务
  worker.postMessage({type: 'xxx', num: 40});
  ```

+ worker.js代码

  ```js
  // worker.js代码

  self.onMessage = e => {
    const {type, num} = e.data;

    if(type === 'xxx') {
      const a = 10;
      self.postMessage(a);
    }
  }
  ```
