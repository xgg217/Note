# 事件处理程序

## Animation.oncancel

+ 在取消动画时触发

  ```js
  addEventListener("cancel", (event) => { })
  oncancel = (event) => { }
  ```

## Animation.onfinish

+ 在动画完成时触发

  ```js
  addEventListener("finish", (event) => { })
  onfinish = (event) => { }
  ```

  ```js
  // Promise形式
  animation.finished.then(() =>
    element.remove()
  )
  ```

## ## Animation.onremove

+ 在动画被移除时触发

  ```js
  addEventListener('remove', (event) => { })
  onremove = (event) => { }
  ```
