## add() 新增

+ API

  ```js
  add(value)
  add(value, key)
  ```

+ 参数

  + value 需要存储的值
  + key 键字用于识别记录。如果未指明，即为空

+ 返回值 一个 IDBRequest 对象，在该操作对象中触发与此相关的后续事件

  ```js
  objectStore.add({ name: "测试", age: 18 }, '用户1')
  ```
