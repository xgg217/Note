# 静态方法

## IDBKeyRange.bound()

+ 指定上界和下界来创建一个新的键范围

  ```
  All keys ≥ x && ≤ y         IDBKeyRange.bound(x, y)
  All keys > x &&< y          IDBKeyRange.bound(x, y, true, true)
  All keys > x && ≤ y         IDBKeyRange.bound(x, y, true, false)
  All keys ≥ x &&< y          IDBKeyRange.bound(x, y, false, true)
  ```

+ 语法

  ```js
  IDBKeyRange.bound(lower, upper)
  IDBKeyRange.bound(lower, upper, lowerOpen)
  IDBKeyRange.bound(lower, upper, lowerOpen, upperOpen)
  ```

+ 参数

  + lower 下限键值
  + upper 上限键值
  + lowerOpen 控制是否排除下限键值
  + upperOpen 控制是否排除上限键值


  ```js
  const range = IDBKeyRange.bound([min1, min2, min3], [max1, max2, max3]);
  // 范围
  // (field1 > min1 || field1=== min1 && field2 > min2 || field1 === min1 && field2 === min2 && field3 >= min3)&& (field1 < max1 || field1=== max1 && field2 < max2 || field1 === max1 && field2 === max2 && field3 <= max3)

  const cursorRequest = dbIndex.openCursor(range, "prev");

  cursorRequest.onsuccess = event => {
    const cursor = event.target.result;

    if (cursor) {
        data.push(cursor.value);

        // 继续游标
        cursor.continue();
      } else {
        // resolve(data);
        // 结束游标
      }
  }

  ```

## IDBKeyRange.only()

+ 指定单个键值来创建一个新的键范围

## IDBKeyRange.lowerBound()

+ 指定结果集的下界来创建一个新的键范围

+ 语法

  ```js
  IDBKeyRange.lowerBound(lower)
  IDBKeyRange.lowerBound(lower, open)
  ```

+ 参数

  + lower 指定新键范围的下限

  + open 可选 指示下限是否排除端点值。默认值为 false

+ 返回值 IDBKeyRange: 新创建的键范围

## IDBKeyRange.upperBound()

+ 指定结果集的上界来创建一个新的键范围
