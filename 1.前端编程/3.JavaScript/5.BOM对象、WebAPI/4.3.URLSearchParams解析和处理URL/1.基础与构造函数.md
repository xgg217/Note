# URLSearchParams 对象

## 概述

+ `URLSearchParams` 对象是浏览器的原生对象，用来构造、解析和处理 URL 的查询字符串（即 URL 问号后面的部分）

## 构造函数

+ 它本身也是一个构造函数，可以生成实例
+ 参数可以为查询字符串，起首的问号?有没有都行，也可以是对应查询字符串的数组或对象

+ 语法

  ```js
  new URLSearchParams()
  new URLSearchParams(options)
  ```

  ```js
  // 方式1 通过 url.search 检索参数，传递到构造函数
  const url = new URL("https://example.com?foo=1&bar=2");
  const params1 = new URLSearchParams(url.search);

  // 方式2 直接从 URL 对象获取 URLSearchParams 对象
  const params1a = url.searchParams;

  // 方式3 传入字符串
  const params2 = new URLSearchParams("foo=1&bar=2");
  const params2a = new URLSearchParams("?foo=1&bar=2");

  // 方式4 传入数组
  var params = new URLSearchParams([['foo', 1], ['bar', 2]]);

  // 方式5 传入对象
  var params = new URLSearchParams({'foo' : 1 , 'bar' : 2});
  ```

+ URLSearchParams会对查询字符串自动编码

  ```js
  // foo的值是汉字，URLSearchParams对其自动进行 URL 编码
  var params = new URLSearchParams({'foo': '你好'});
  params.toString() // "foo=%E4%BD%A0%E5%A5%BD"
  ```
