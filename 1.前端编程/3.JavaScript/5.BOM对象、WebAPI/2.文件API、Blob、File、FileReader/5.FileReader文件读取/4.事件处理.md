# 事件处理

## 概述

+ 使用 addEventListener() 方法或通过将事件侦听器分配给此接口的 oneventname 属性来侦听这些事件
+ 一旦不再使用 FileReader，请使用 removeEventListener() 删除事件侦听器，以避免内存泄漏

## 事件

+ abort 当读取被中止时触发

  + 读取中断或调用 `reader.abort()` 方法时触发

+ error 当读取由于错误而失败时触发

+ load 读取成功完成时触发

+ loadend 读取完成时触发，无论成功与否

  + 读取完成后触发，不管是否成功
  + 触发顺序排在 `onload` 或 `onerror` 后面

+ loadstart 读取开始时触发

+ progress 读取数据时定期触发

  + 该事件在读取**Blob**时触发（读取过程中周期性触发）


## 示例

+ 用于中止文件上传

  ```js
  var reader = new FileReader();
  reader.abort();
  ```

+ 主要用来显示读取进度

  ```js
  reader.onprogress = function(e){
    if(e.lengthComputable){
        progress = e.loaded / e.total
    }
    console.log(e, progress);
  }
  ```
