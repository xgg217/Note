# 函数

## API

+ `@func [<FunctionName>]` 标识一个函数

  ```js
  /**
   * @func
   */
  function foo() {
    ...
  }
  ```

## 声明方式

+ 方式1

  ```JS
  /** @type {function} */
  ```

  ```js
  // 一个包含参数的函数
  /** @type {function(number, string)} */
  ```

  ```js
  // Object结构的参数
  /** @type {function({ arg1: number, arg2: string })} */

  /** @type {(s:string, b:boolean) => number} */
  ```

  ```js
  // 一个包涵参数和返回值的函数
  /** @type {function(number, string): boolean} */
  ```
