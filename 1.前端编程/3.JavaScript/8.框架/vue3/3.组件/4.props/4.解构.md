# 解构

## 问题

+ 直接解构：响应式会丢失

  ```js
  const { count } = defineProps<{ count: number }>();
  ```

## 方式1 props.x

+ 代码

  ```js
  const props = defineProps<{ count: number }>();

  props.count
  ```

## 方式2 toRefs

+ 代码

  ```js
  const props = defineProps<{ count: number }>();

  const { count } = toRefs(props);
  ```


## vue3.5

+ 直接一步到位设置默认值

  ```html
  <template>
    <div>
      {{ testCount }}
    </div>
  </template>

  <script setup>
    import { defineProps } from 'vue';
    const { testCount=18 } = defineProps({
      testCount: {
        type: Number,
      },
    });
  </script>
  ```
