# 国际化处理

## 概述

+ 本地化优先于技术方案

+ 文案内容改变

  + **固定内容**

    + 后端模板
    + 映射表，不同区域读取不同的内容
    + i18n等相关工具库

      + vue-i18n
      + react-intl
      + i18next

    + 翻译方式

      + 人工翻译
      + 工具辅助
      + 翻译平台

  + 动态数据

    + 后端数据库批量翻译存入数据库

      + 同样结构不同语言内容的表
      + 接口添加标准地区语言配置区分获取不同数据

    + 不同区域是不同的数据库

    + 固定内容可以响应式切换，动态数据内容的切换

      + 切换语言环境后，刷新页面重新请求
      + 后端直接返回所有语言环境数据，由前端控制渲染

  + 本地化的 货币、数字、日期

+ 样式和外观

  + 文案

    + 内容长度
    + 字体大小，间隔，对齐方式
    + 方向

  + 图片,颜色等

+ 图片内容改变

  + 重要位置和活动图片
  + 常规图片与icon

    + 常规图片和icon的切换
    + 常规图片中显示的文案，活动内容，价格...

+ 本地化内容的投放

  + 前端请求后根据语言标识获取数据
  + 后端根据语言标识读取相应数据内容并输出
