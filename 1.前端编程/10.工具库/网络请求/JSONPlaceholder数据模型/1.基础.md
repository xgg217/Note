# JSONPlaceholder数据模型

## 概述

+ 官网 https://jsonplaceholder.typicode.com/

+ JSONPlaceholder 是一个在线 REST API 服务，提供一系列模拟数据接口，适合用于：

  + 前端开发阶段的接口调试；
  + HTTP 请求库（如 Axios、Fetch）的测试；
  + 后端框架或工具的 API 调用演示；
  + 教学与 Demo 示例。

+ 它的数据内容不会变化，也不会真的存入数据库，因此可以无限次请求，完全免费、无需注册

## 可用的资源列表

+ JSONPlaceholder 模拟了典型的 RESTful API 结构，包含以下资源：

  ```md
  资源名       示例接口            说明
  Posts        /posts         模拟文章或博客内容
  Comments    /comments       模拟评论数据
  Albums      /albums         模拟相册数据
  Photos      /photos         模拟图片数据
  Todos        /todos         模拟待办事项
  Users        /users         模拟用户信息
  ```

## 支持的 HTTP  方法

+ 这些请求方法都可以正常返回对应的结果，方便你在开发阶段验证前端 CRUD 功能

  ```md
  方法         描述
  GET          获取数据
  POST         模拟创建数据
  PUT          模拟替换数据
  PATCH        模拟局部更新
  DELETE       模拟删除数据
  ```

## 字段

+ Users 用户实体模型 用户是JSONPlaceholder数据模型的核心基础，每个用户包含以下关键字段：

  + id: 唯一用户标识符
  + name: 用户全名
  + username: 登录用户名
  + email: 电子邮箱地址
  + address: 详细地址信息（包含街道、城市、邮编等）
  + phone: 联系电话
  + website: 个人网站
  + company: 公司信息

+ Posts 文章数据关系 帖子是用户发布的内容，每个帖子都与特定用户关联：

  + userId: 关联的用户ID
  + id: 帖子唯一标识
  + title: 帖子标题
  + body: 帖子正文内容

+ Comments 评论系统架构 评论是用户对帖子的回复，建立了完整的内容互动关系：

  + postId: 关联的帖子ID
  + id: 评论唯一标识
  + name: 评论者姓名
  + email: 评论者邮箱
  + body: 评论内容

+ ‌Albums 相册

  + userId: 关联的用户ID
  + id: 帖子唯一标识
  + title: 标题

+ Todos 待办事项

  ```md
  "userId": 1,
  "id": 1,
  "title": "delectus aut autem",
  "completed": false
  ```

+ Photos 图片

  ```md
  "albumId": 1,
  "id": 1,
  "title": "accusamus beatae ad facilis cum similique qui sunt",
  "url": "https://via.placeholder.com/600/92c952", 原图
  "thumbnailUrl": "https://via.placeholder.com/150/92c952" 缩略图
  ```

## 基础 URL

+ https://jsonplaceholder.typicode.com/

## 请求示例

+ 获取所有文章

  ```js
  GET https://jsonplaceholder.typicode.com/posts

  // 返回示例：
  [
    {
      "userId": 1,
      "id": 1,
      "title": "sunt aut facere repellat provident occaecati",
      "body": "quia et suscipit..."
    },
    ...
  ]
  ```

+ 获取单个文章详情

  ```js
  GET https://jsonplaceholder.typicode.com/posts/1

  // 响应示例：
  {
    "userId": 1,
    "id": 1,
    "title": "sunt aut facere repellat provident occaecati",
    "body": "quia et suscipit..."
  }
  ```

+ 提交一条新文章

  ```js
  POST https://jsonplaceholder.typicode.com/posts
  Content-Type: application/json

  {
    "title": "Hello JSONPlaceholder",
    "body": "这是一篇测试文章",
    "userId": 1
  }

  // 虽然返回了新 ID，但不会真的写入服务器，刷新后仍是原始数据
  {
    "id": 101,
    "title": "Hello JSONPlaceholder",
    "body": "这是一篇测试文章",
    "userId": 1
  }
  ```

+ 筛选与查询参数

  ```md
  获取 userId 为 1 的所有文章：

  GET https://jsonplaceholder.typicode.com/posts?userId=1
  ```
