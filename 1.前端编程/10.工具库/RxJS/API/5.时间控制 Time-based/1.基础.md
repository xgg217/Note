# 时间控制 Time-based

## 操作符

+ 操作符

  | 操作符             | 行为                             | 典型场景         |
  | ------------------ | -------------------------------- | ---------------- |
  | `debounceTime(ms)` | 安静 N ms 后放行最后一个值       | 搜索输入防抖     |
  | `throttleTime(ms)` | 每 N ms 最多放行一个值           | 滚动监听、resize |
  | `delay(ms)`        | 每个值延迟 N ms 后发出           | 动画、延迟显示   |
  | `auditTime(ms)`    | 每 N ms 发出该周期内最后一个值   | 性能监控采样     |
  | `sampleTime(ms)`   | 每 N ms 采样源流的最新值         | 固定频率采样     |
  | `bufferTime(ms)`   | 积攒 N ms 的值一次性发出（数组） | 批量日志上报     |
