# 基础

## 操作符

+ 操作符

  | 操作符                    | 用途                                     | 典型场景             |
  | ------------------------- | ---------------------------------------- | -------------------- |
  | `merge(a$, b$)`           | 合并多条流，谁来了就发谁                 | 来自多个按钮的事件   |
  | `concat(a$, b$)`          | 串接流：前一条完了才订阅后一条           | 按顺序执行多个请求   |
  | `combineLatest([a$, b$])` | 任一流更新时，取所有流的最新值           | 表单联动、多条件筛选 |
  | `forkJoin([a$, b$])`      | 等所有流 complete，取每条流的最后值      | 并行请求等全部完成   |
  | `zip(a$, b$)`             | 按位置一一配对                           | 严格的一对一对应     |
  | `race(a$, b$)`            | 取最先发出值的流，丢弃其余               | 超时竞争、多源取最快 |
  | `withLatestFrom(b$)`      | 主流发值时，携带另一条流的最新值         | 主操作关联上下文数据 |
  | `startWith(value)`        | 给流一个初始值（在第一个真实值之前发出） | 设置默认状态         |
