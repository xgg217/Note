# 基础

## 概述

+ 转换操作符把流中的值从一种形状变成另一种形状

## 操作符

+ 操作符

  | 操作符           | 用途                               | 典型场景                          |
  | ---------------- | ---------------------------------- | --------------------------------- |
  | `map(fn)`        | 每个值经过函数转换                 | 提取字段、格式化                  |
  | `scan(fn, seed)` | 累积计算（像 reduce 但每步都输出） | 累加器、状态聚合                  |
  | `tap(fn)`        | **不改值**，只"看一眼"或执行副作用 | 调试日志、埋点、更新 loading 状态 |
