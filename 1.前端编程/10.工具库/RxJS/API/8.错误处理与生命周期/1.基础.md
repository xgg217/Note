# 基础

## 操作符

+ 操作符

  | 操作符                  | 用途                                     | 典型场景             |
  | ----------------------- | ---------------------------------------- | -------------------- |
  | `catchError(fn)`        | 拦截错误，返回替代流                     | 请求失败给默认值     |
  | `retry(n)`              | 失败后重试 N 次                          | 网络抖动自动重试     |
  | `retry({count, delay})` | 自定义重试策略（含延迟/指数退避）        | 指数退避重试         |
  | `timeout(ms)`           | 超时报错                                 | 请求超时控制         |
  | `finalize(fn)`          | 终止时执行（complete/error/unsubscribe） | 清理资源、关 loading |
