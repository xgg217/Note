# 文本

## ctx.fillText 绘制填充文本

+ `ctx.fillText( textStr, x, y [, maxWidth] )`

  + textStr 文本内容
  + x , y 文本位置(坐标)
  + maxWidth (可选) 设置文本最大宽度

    + 如果文本宽度 > 最大宽度 就会放缩， 压缩在maxWidth范围内

  ```js
  ctx.fillText("绘制填充文本", 100, 100);
  ```
