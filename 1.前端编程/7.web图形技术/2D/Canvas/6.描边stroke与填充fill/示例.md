## 示例1

+ 示例1

  ```js
  /** @type {HTMLCanvasElement} */
  const canvas1 = document.querySelector(".c1");

  /** @type {CanvasRenderingContext2D} */
  const ctr = canvas1.getContext("2d");

  ctr.fillStyle = "red";
  ctr.strokeStyle = "blue";
  ctr.lineWidth = 10;

  ctr.rect(100,100,200,100);
  ctr.fill();
  ctr.stroke();
  ```
