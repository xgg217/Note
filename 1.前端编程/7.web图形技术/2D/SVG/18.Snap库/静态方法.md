# 静态方法

## 概述

+ Snap除了使用工厂函数以外， 还提供了许多的静态方法(工具方法)

## Snap.sin(角度)

+ Snap.sin(角度) ：直接对角度进行计算
+ 与 `Math.sin()` 不同， 支持的是弧度

  ```js
  //snap库的三角函数计算
  Sanp.sin(90) == 1

  //Math的三角函数计算,需要将角度转换成弧度才能计算。
  Math.sin(90 * Math.PI / 180) == 1
  ```

## Snap.deg(rad)

+ Snap.deg(rad) 弧度转角度

## Snap.rad(deg)

+ Snap.rad(deg) 角度转弧度
