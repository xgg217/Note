# 基础 path

## 概述

+ 根据点连接，绘制各种图形

  + 支持直线（线段，折线，矩形，多边形）
  + 曲线（贝塞尔曲线，弧线）

+ path 元素是用来定义形状的通用元素
+ 所有的基本形状都可以用 path 元素来创建

  ```html
  <svg viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg">
    <path
      d="M 10,30
        A 20,20 0,0,1 50,30
        A 20,20 0,0,1 90,30
        Q 90,60 50,90
        Q 10,60 10,30 z" />
  </svg>
  ```
