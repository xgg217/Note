# 基础

## 概述

+ 使用 `<filter>` 标签定义滤镜（容器），并且置于 `<defs>` 区块中

  + 可以包含多个具体的滤镜（效果）

+ 图形使用 `filter` 属性引用滤镜

  ```html
  <svg width="400" height="400" view-box="0 0 100 100">
    <defs>
      <filter id="f1">
        <feGaussianBlur stdDeviation="2" />
      </filter>
    </defs>

    <rect x="10" y="10" width="50" height="50" fill="#f00" filter="url(#f1)"></rect>
  </svg>
  ```

## 滤镜生效方式

+ SVG 方式

  ```html
  <rect filter="url(#f1)"></rect>

  <!-- 或者 -->
  <circle style="filter: url(#f1)" />
  ```

+ CSS 方式

  ```css
  rect {
    filter: url(#f1);
  }
  ```
