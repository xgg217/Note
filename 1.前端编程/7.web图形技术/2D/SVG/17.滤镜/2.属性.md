# 属性

## 通用属性 in

+ 当某一个滤镜作用在图像上，不会立刻生效，而是会将处理结果存入缓存

  + 可以使用 result 属性定义缓存名称

  ```html
  <defs>
    <filter id="f1">
      <feGaussianBlur stdDeviation="2" result="r1" />
    </filter>
  </defs>
  ```

+ 下一个滤镜可以使用 `in` 属性，来将缓存中的结果加入当前滤镜处理中

  ```html
  <defs>
    <filter id="f1">
      <feGaussianBlur stdDeviation="2" result="r1" />
      <feOffset in="r1" dx="2" dy="2" result="r1" />
    </filter>
  </defs>
  ```

+ 如果没有对装入缓存的结果命名，下一个滤镜会默认装载上一个相邻的处理结果

  ```html
  <defs>
    <filter id="f1">
      <feGaussianBlur stdDeviation="2" />
      <feOffset dx="2" dy="2" />
    </filter>
  </defs>
  ```

+ 也可以使用 in 属性，指定对原图片做处理 `SourrceGraphic`

  ```html
  <defs>
    <filter id="f1">
      <feGaussianBlur stdDeviation="2" />
      <feOffset in="SourrceGraphic" dx="2" dy="2" />
    </filter>
  </defs>
  ```

## 专有属性

+ `id` 唯一标识滤镜，方便图形引用
+ `x, y, width, height` 控制滤镜的作用范围

  + 默认 `x = y = -10%` `width = height = 120%`

  ```html
  <defs>
    <filter
      id="MyFilter"
      filterUnits="userSpaceOnUse"
      x="0"
      y="0"
      width="200"
      height="120">
    </filter>
  </defs>
  ```

+ filterUnits 设置 filter 区域的数据单元

  + objectBoudingBox *默认值* 基于图形使用百分比
  + userSpaceOnUse 具体值，基于坐标系

  ```html
  <defs>
  <filter id="f1" filter-units="objectBoudingBox">
    <feGaussianBlur stdDeviation="2" />
  </filter>
  ```

+ primitiveUnits 设置 filter 容器内部具体滤镜区域的数据单元

  + objectBoudingBox  基于图形使用百分比
  + userSpaceOnUse *默认值* 具体值，基于坐标系
