# 矩阵运算

## 方式1 矩阵和单独数字的运算

+ 单独数字

  ```js
  mat4 m=mat4(
      1,5,9,13,
      2,6,10,14,
      3,7,11,15,
      4,8,12,16
  );
  m+=1.0;
  m-=1.0;
  m*=2.0;
  m/=2.0;
  ```

## 方式2 矩阵和矩阵的运算

+ 矩阵和矩阵之间可以进行加减乘除

  ```js
  mat4 m=mat4(
      2,16,8,8,
      4,8,8,8,
      8,4,8,8,
      16,8,8,8
  );
  mat4 n=mat4(
      1,4,1,2,
      2,4,2,1,
      4,4,1,2,
      8,4,2,1
  );
  ```

+ 矩阵加法：相同索引位置的元素相加

  ```js
  m+=n;

  /*
      3, 20, 9, 10,
      6, 12, 10, 9,
      12, 8, 9, 10,
      24, 12, 10, 9,
  */
  ```

+ 矩阵减法：相同索引位置的元素相减

  ```js
  m-=n;
  /*
      1, 12, 7, 6,
      2, 4, 6, 7,
      4, 0, 7, 6,
      8, 4, 6, 7,
  */
  ```

+ 矩阵乘法

  ```js
  m/=n;
  /*
      2, 4, 8, 4,
      2, 2, 4, 8,
      2, 1, 8, 4,
      2, 2, 4, 8,
  */
  ```

+ 矩阵除法：相同索引位置的元素相除

  ```js
  m*=n;
  /*
      58, 68, 64, 64,
      52, 80, 72, 72,
      64, 116, 88, 88,
      64, 176, 120, 120,
  */
  ```
