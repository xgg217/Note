# 实际应用

## 生成随机颜色

+ code

  ```js
  /**
   * 生成随机颜色（使用HSL更易控制鲜艳程度）
   * @param {饱和度} saturation
   * @param {亮度} luminosity
   * @returns
   */
  export const getRandomColor = (saturation = 0.7,luminosity = 0.5)=>{
    return new THREE.Color().setHSL(Math.random(), saturation, luminosity);
  }
  ```
