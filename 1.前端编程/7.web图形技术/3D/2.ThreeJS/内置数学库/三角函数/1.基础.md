# 三角函数

## 定义

+ 在直角坐标系中，已知：

  + 射线OP

  + 点O 是射线OP的起点，也是直角坐标系的原点

  + 点P 是射线OP 上的任意一点

  + 点P 的位置(x,y)

  + 点M(x,0)在X 轴上

  + PM⊥OM

  + ∠MOP=α

  + OM = x

  + PM = y

  + |OP| = sqrt(x²+y²) >= 0


  ![alt text](images/三角函数.png)

+ 根据三角函数的定义可知：

  ```js
  sin α=y/r
  cos α=x/r
  tαn α=y/x
  cos α=x/y
  ```

## 注意

+ 注：关于∠α 的单位问题

+ α后面有角度符号°时，那α 就是角度。否则，就是弧度

+ 在角度和弧度做运算时，需做单位的换算，要么都统一为弧度，要么都统一为角度
