# 角度

## degToRad

+ `.degToRad ( degrees : Float ) : Float` 将度转化为弧度

## radToDeg

+ `.radToDeg ( radians : Float ) : Float` 将弧度转换为角度

  ```js
  const degrees = 90;
  const radians = THREE.MathUtils.degToRad(degrees); // 转换为弧度值
  const convertedDegrees = THREE.MathUtils.radToDeg(radians); // 转换回角度值
  ```
