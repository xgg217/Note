# 条件判断

## if-else 语句

+ 单独使用 `if`

  ```js
  float x = 10.0

  if(x > 100.0){
    x = 100.0;
  }
  ```

+ `if-else` 形式

  ```js
  float x = 10.0

  if(x > 100.0){
    x = 100.0;
  } else {
    x = x + 1.0;
  }
  ```

