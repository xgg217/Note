# 单位向量

## 概述

+ 单位向量是向量长度`.length()`为1的向量

  ![向量表示速度](../images/向量表示速度.jpg)

  ```js
  // 单位向量v
  const v = new THREE.Vector3(1,0,0);
  console.log('向量长度',v.length()); // 1
  ```

  ```js
  // 非单位向量dir
  const dir = new THREE.Vector3(1,1,0);
  console.log('向量长度',v.length()); // 1.4142135623730951
  ```
