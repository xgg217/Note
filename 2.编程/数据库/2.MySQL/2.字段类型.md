# 字段类型

## bit

+ 占 1 位
+ `0` 或 `1`
+ `false` 或 `true`

## int 整数

+ 占 32 位

## decimal(M,N) 小数

+ 能精确计算的实数
+ M：总的数字位数
+ N：小数位数

  ```
  3.14159

  M --> 6
  N --> 5
  ```

## char(n) 字符串

+ 固定长度位的字符

## varchar(n) 字符串

+ 长度可变
+ 最大长度位n的字符

## text 字符串

+ 大量的字符

## data

+ 仅日期

## time

+ 仅时间

## datetime

+ 日期和时间

长整数 bigint(21) 可以设置长度
浮点数（小数） float
双精度浮点数 double
