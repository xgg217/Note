# 进制

## 概述

+ Python中，除了10进制还有二进制、八进制、十六进制

## 2进制

+ 1,2
+ `0b` 或者 `0B` 开头

  ```py
  # 二进制
  0b10 # 十进制的 2
  ```

## 8进制

+ 1,2,3,4,5,6,7,10,11,12,13,14,15,16,17,20
+ `0o` 或者 `0O` 开头

  ```py
  0o10 # 10进制的 8
  ```

## 16进制

+ 1,2,3,4,5,6,7,8,9,A,B,C,D,E,F,10,11...1F,20
+ `0x` 或者 `0X` 开头

  ```py
  0x1f # 10进制的 31
  ```
