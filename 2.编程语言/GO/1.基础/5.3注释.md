# 注释

## 概述

+ 单行注释， `//`
+ 多行注释， `/**/`

+ 快捷键：选中代码 然后点击键盘的 `contrl + ?`

  ```go
  package main

  import "fmt"
  func main() {
      // 单行注释
      /*多行注释
      fmt.Println(666)
      fmt.Println(6 + 9)
      fmt.Println(6 - 9)
  fmt.Println(6* 9)
      fmt.Println(16 / 9) // 商
      */
      fmt.Println(16 % 9) // 余数
  ```
