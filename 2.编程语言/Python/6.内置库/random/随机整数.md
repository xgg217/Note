# 随机整数

## random.randint()

+ `randint(a,b)` 生成一个 `[a,b]` 之间的整数

# random.getrandbits()

+ `getrandbits(k)` 生成一个 `k` 比特长度的随机整数

## random.randrange()

+ 效率最高 **推荐使用**
+ `randrange(start, stop[, step])` 生成一个 `[start, stop)` 之间以 step 为步长的随机整数

  ```py
  import random

  random.randrange(0, 100, 4)
  # 96
  ```
