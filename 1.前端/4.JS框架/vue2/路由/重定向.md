# 重定向

## 使用

*   重定向也是通过 routes 配置来完成，下面例子是从 /a 重定向到 /b

    ```javascript
    const router = new VueRouter({
      routes: [
        {
          path: '/a',
          redirect: '/b'
        }
      ]
    })
    ```

*   重定向的目标也可以是一个命名的路由：

    ```javascript
    const router = new VueRouter({
      routes: [
        { path: '/a', redirect: { name: 'foo' }}
      ]
    })
    ```

*   甚至是一个方法，动态返回重定向目标：

    ```javascript
    const router = new VueRouter({
      routes: [
        { path: '/a', redirect: to => {
          // 方法接收 目标路由 作为参数
          // return 重定向的 字符串路径/路径对象
        }}
      ]
    })
    ```
