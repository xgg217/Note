# 条件渲染

## 方法1：利用 display: none

*   适用场景：组件方法切换

    ```react&#x20;jsx
    return (
      <div>
        <h1 style={{ display: this.this.state.visible ? 'block' : 'none' }} >标题</h1>
        <button onClick={ () => {
          this.setState((state) => {
            return {
              state: !state.state
            }
          })
        } }>
        </button>
      </div>
    )
    ```

## 方法2：利用 null

*   使用场景：

    ```react&#x20;jsx
    const h1 = this.state.visible ? (<h1>标题</h1>) : null

    return (
      <div>
        { h1 }
        <button onClick={ () => {
          this.setState((state) => {
            return {
              state: !state.state
            }
          })
        } }>
        </button>
      </div>
    )
    ```
