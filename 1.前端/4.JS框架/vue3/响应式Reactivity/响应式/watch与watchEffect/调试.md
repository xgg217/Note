# 调试

## 概述

*   和 `computed()` 类似，侦听器也支持 `onTrack` 和 `onTrigger` 选项

    ```javascript
    watch(source, callback, {
      onTrack(e) {
        debugger
      },
      onTrigger(e) {
        debugger
      }
    })

    watchEffect(callback, {
      onTrack(e) {
        debugger
      },
      onTrigger(e) {
        debugger
      }
    })
    ```

## 注意

*   侦听器的 `onTrack` 和 `onTrigger` 选项仅会在开发模式下工作
