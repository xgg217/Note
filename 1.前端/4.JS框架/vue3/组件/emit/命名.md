# 命名

## 命名方式

  - `camelCase` 或 `PascalCase`

    ```javascript
    this.$emit('myEvent')
    ```

  - `kebab-case` **推荐**

    ```javascript
    this.$emit('my-event')
    ```

## 理由

  - 不同于组件和 `prop`，事件名不会被用作一个 `JavaScript` 变量名或 `property` 名，所以就没有理由使用 `camelCase` 或 `PascalCase` 了。

  - 并且 `v-on` 事件监听器在 `DOM` 模板中会被自动转换为全小写 (因为 `HTML` 是大小写不敏感的)，所以 `@myEvent` 将会变成 `@myevent` ——导致 `myEvent` 不可能被监听到。
