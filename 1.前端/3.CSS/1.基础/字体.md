# 字体

## 设置字体

  - font-family

    ```javascript
    font-family: consolas,微软雅黑,Arial,sans-serif;
    ```

## sans-serif

  - 非衬线字体

## web 字体

  - 解决用户电脑上没有安装相应字体，强制让用户下载该字体

  - 使用 `@font-face` 指令制作一个新字体

    ```css
    /* index.css */
    @font-face {
      font-family: "good night";
      src: url("./src/晚安体.ttf");
    }

    p {
      font-family: "good night";
    }
    ```
