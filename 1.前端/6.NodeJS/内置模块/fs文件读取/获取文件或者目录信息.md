# 获取文件或者目录信息

## 概述

  - 获取文件或目录信息

## fsPromises.stat(path\[, options])

  - 参数

      - path

      - options

## 判断是否是目录 stat.isDirectory()

  - 判断是否是目录 `stat.isDirectory()`

    ```javascript
    const path = require("path");
    const fs = require("fs")

    const filename = path.resolve(__dirname, "./files");

    try {
      const rf =  await fs.promises.stat(filename);
      console.log(rf);

      rf.isDirectory(); // true
      rf.isFile(); // false
    } catch (error) {
      console.log(error);
    }
    ```

## 判断是否是文件 stat.isFile()

  - 判断是否是文件 `stat.isFile()`

    ```javascript
    const fsPromises = require('fs/promises');

    fsPromises.stat('./b.txt').then(res => {
      console.log(res);
      res.isDirectory(); // true
      res.isFile(); // false
    }).catch(err => {
      console.error(err);
    });
    ```
