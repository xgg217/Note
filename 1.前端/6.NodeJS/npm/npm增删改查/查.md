# 查

## 搜索模块

  - 命令

    ```bash
    npm search express
    ```

## 查看所有全局安装的模块

  - 查看所有全局安装的模块

    ```bash
    npm list [-g] [--depth=深度依赖]
    ## list 别名 : ls la ll

    ## 例如
    npm ls
    npm la -g

    ## 查看0级依赖
    npm la --depth=0

    ## 查看1级依赖
    npm la --depth=1
    ```

  - 如果要查看某个模块的版本号

    ```javascript
    npm list grunt
    ```

## 查看包信息

  - 可以查看所有包信息，没有安装的可也可看到

    ```bash
    npm view 包名 [子信息]
    ## view 别名 : v info show

    ## 例如
    npm v react
    npm v react name
    ```
