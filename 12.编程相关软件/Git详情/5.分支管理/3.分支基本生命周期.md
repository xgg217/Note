# 生命周期

## 周期1-新建分支

+ 新建分支

  ```text
  feature/*     - 新功能开发
  bugfix/*      - 缺陷修复
  hotfix/*      - 紧急生产修复
  release/*     - 版本发布
  chore/*       - 维护性任务
  docs/*        - 文档更新
  refactor/*    - 代码重构
  ```

## 周期2-开发

+ 在分支上开发测试 可能产生多次提交

## 周期3-合并

+ 合并到主分支

## 周期4：测试/部署

+ 主分支测试/部署

## 周期5

+ 删除开发分支
