# 文档注释

## 书写方式1

*   代码

    ```javascript
     /**
     * 函数求和
     * @param {number} a 第一个数字
     * @param {number} b 第二个数字
     * @returns {number} 返回相加的结果
     */
    function sum(a, b) {
      return a + b;
    }

    sum()
    ```

## 书写方式2

*   代码

    ```javascript
    /**
     * 函数求和
     * @param {number} a 第一个数字
     * @param {number} b 第二个数字
     * @returns {number} 返回相加的结果
     */
    const sum = function sum(a, b) {
      return a + b;
    }

    sum()
    ```
