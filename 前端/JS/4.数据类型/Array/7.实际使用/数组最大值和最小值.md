# 数组最大值和最小值

## 数组中的最大值

  - 结合使用 `Math.max()` 与 展开运算符( `...` )，获取数组中的最大值

    ```javascript
    const arrayMax = arr => Math.max(...arr);
    // arrayMax([10, 1, 5]) -> 10
    ```

## 数组中的最小值

  - 结合使用 `Math.max()` 与 展开运算符( `...` )，获取数组中的最小值

    ```javascript
    const arrayMin = arr => Math.min(...arr);
    // arrayMin([10, 1, 5]) -> 1
    ```
