# ??=逻辑空赋值

## 概述

*   逻辑空赋值运算符 (`x ??= y`) 仅在 `x` 是 `null` 或 `undefined` 时对其赋值

## 示例

*   示例1

    ```javascript
    const a = { duration: 50 };
    a.duration ??= 10;
    console.log(a.duration); // 50

    a.speed ??= 25;
    console.log(a.speed); // 25

    console.log(a); // { duration: 50, speed: 25 }
    ```
