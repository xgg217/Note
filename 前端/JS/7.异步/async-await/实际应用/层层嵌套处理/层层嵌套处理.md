# 层层嵌套处理

## 层层嵌套

*   下一个回调函数的参数依赖于上一个异步的返回结果

*   `async` 函数 返回值是 `Promise`。

    ```javascript
    async function read(url) {
      const val1 = await readFile(url);
      const val2 = await readFile(val1);
      const val3 = await readFile(val2);
      return val3;
    }

    read('./data.num.txt')
      .then((val3) => {
        console.log(val3);
      })
      .catch(err => {
        console.log(err);
      });
    ```
