# 基础

## 错误分类

*   语法错误:会导致整个脚本块无法执行

*   运行错误

    *   运行报错:会导致当前脚本块后续代码无法执行

    *   运行结果不符合预期

## 调试错误

*   控制台打印

*   断点调试（`debugger`）

## 抛出错误

*   错误再 js 中本质是一个对象，抛出一个错误的语法为

    ```javascript
    throw 错误对象;
    ```

*   错误对象的构造函数为 Error

    ```javascript
    throw new Error('出错了');
    ```
