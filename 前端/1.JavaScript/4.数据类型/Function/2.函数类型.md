# 函数类型

## typeof

  - 使用 `typeof`

    ```js
    const myFunc = () => {};

    console.log(typeof myFunc); // 'function'
    ```

## 检测构造函数

  - 使用 `instanceof`

      ```js
      class Person{
        constructor(name) {
          this.name = name;
        }
      }

      const person = new Person('小刚刚');

      console.log(person instanceof Person); // true
      console.log(person instanceof Object); // true
      ```
