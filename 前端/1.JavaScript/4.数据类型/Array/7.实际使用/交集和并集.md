# 交集和并集

## 两个集合的并集

+ 代码

    ```js
    export const union = (a, b) => {
      var newArr = a.concat(b);
      return this.unique(newArr);
    }
    ```

## 两个集合的交集

+ 代码

    ```js
    export const intersect = (a, b) => {
      var _this = this;
      a = this.unique(a);
      return this.map(a, function (o) {
          return _this.contains(b, o) ? o : null;
      });
    }
    ```
