# 基础

## 数组的定义

  - 数组（array）是按次序排列的一组值。每个值的位置都有编号（从0开始），整个数组用方括号表示

    ```js
    var arr = ['a', 'b', 'c'];
    ```

## 数组的本质

  - 本质上，数组属于一种**特殊的对象**

  - `typeof` 运算符会返回数组的类型是 `object`

  - 数组的特殊性体现在，它的键名是按次序排列的一组整数（0，1，2…）

    ```js
    var arr = ['a', 'b', 'c'];

    Object.keys(arr)
    // ["0", "1", "2"]
    ```
