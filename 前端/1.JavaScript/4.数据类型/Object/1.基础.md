# 基础

## ES5

  - 代码

    ```js
    const obj = {
      name: name,
      sayName: function() {
        console.log(this.name);
      }
    };
    ```

## ES6

  - 代码

    ```js
    const name = 'xgg';
    const obj = {
      name, // 当 键 和 值 名字一样时可以简写
      sayName() { // 函数简写
        console.log(this.name);
      }
    };
    ```
