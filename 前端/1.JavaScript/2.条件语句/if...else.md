# if 与 if...else

## if

+ `if` 结构先判断一个表达式的布尔值，然后根据布尔值的真伪，执行不同的语句

    ```js
    if (m === 3) {
      m += 1;
    }
    ```

## if 判断

+ 优先采用“严格相等运算符”（`===`），而不是“相等运算符”（`==`）

## if...else

+ `if` 代码块后面，还可以跟一个 `else` 代码块，表示不满足条件时，所要执行的代码

    ```js
    if (m === 3) {
      // then
    } else {
      // else
    }
    ```

+ 多个 `if...else` 语句可以连写在一起

    ```js
    if (m === 0) {
      // ...
    } else if (m === 1) {
      // ...
    } else if (m === 2) {
      // ...
    } else {
      // ...
    }
    ```
