# 函数

## 函数参数的解构赋值

- 数组

    ```js
    function add([x, y]){
      console.log(x); // 1
      console.log(y); // 2
      return x + y;
    }
    add([1, 2]); // 3
    ```

- 对象

    ```js
    function sun({x, y}){
      console.log(x); // 1
      console.log(y); // 2
      return x + y;
    }

    sun( { x:1, y: 2 } );
    ```

## 参数 默认值

- 代码

    ```js
    function move({x = 0, y = 0} = {}) {
      return [x, y];
    }

    move({x: 3, y: 8}); // [3, 8]
    move({x: 3}); // [3, 0]
    move({}); // [0, 0]
    move(); // [0, 0]
    ```
