# 可迭代协议

## 可迭代协议

  - ES6规定：如果一个对象具有智能符号的属性 `Symbol.iterator`，并且属性值是一个迭代器创建函数，则该对象是可迭代的（`iterable`）

    ```js
    const obj = {
      [Symbol.iterator]() {
        return {
          next() {
            return {
              value: ?,
              done: boolean
            }
          }
        }
      }
    }
    ```

## 展开运算符与可迭代对象

  - 展开运算符可以作用于可迭代对象，这样就可以轻松将可迭代对象转换为数组
