# 背景图片

## 设置背景图片

  - 代码

    ```html
    <div :style="url"></div>
    ```

    ```js
    return {
      url: {
        backgroundImage: "url(" + require("@/assets/images/cs/1.jpg") + ") ",
        backgroundPosition: "center center",
        backgroundRepeat: "no-repeat",
        backgroundSize: "100% auto",
      },
    }
    ```
