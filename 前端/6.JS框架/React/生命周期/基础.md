# 基础

## 生命周期定义

+ 组件从诞生到销毁会经历一系列的过程

## React 类组件 生命周期

+ React 在组件的生命周期中提供了一系列的钩子函数（类似事件），可以让开发者在函数中注入代码，这些代码会在适当的时候运行

+ 生命周期仅仅存在**类**组件中

+ 函数组件每次调用都是重新运行函数，旧的组件即刻被销毁
