# 定义

## 概述

  - 枚举会出现在**编译结果**中，编译结果中表现为一个**对象**

## 定义

  - 定义枚举

    ```javascript
    enum 枚举名{
      枚举字段1 = 值1,
      枚举字段2 = 值2,
      枚举字段3 = 值3,
      ...
    }
    ```

    ```javascript
    enum Gender{
      male = '男',
      female = '女'
    }

    // 赋值
    let gender:Gender = Gender.male;
    console.log(gender); // '男'

    // 遍历枚举对象
    function printGenders(obj:object) {
      const val = Object.values(obj);
      console.log(val); // [ '男', '女' ]
    }

    printGenders(Gender);
    ```
