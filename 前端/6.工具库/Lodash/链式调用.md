# 链式调用

## chain

- 创建一个lodash包装实例，包装value以启用显式链模式。要解除链必须使用 \_#value 方法

    ```js
    var users = [
      { 'user': 'barney',  'age': 36 },
      { 'user': 'fred',    'age': 40 },
      { 'user': 'pebbles', 'age': 1 }
    ];

    var youngest = _
      .chain(users)
      .sortBy('age')
      .map(function(o) {
        return o.user + ' is ' + o.age;
      })
      .head()
      .value();
    // => 'pebbles is 1'
    ```
