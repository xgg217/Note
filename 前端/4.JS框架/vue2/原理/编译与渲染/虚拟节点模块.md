# 虚拟节点模块

## 概述

- 提供一个函数 `createVNode`，根据提供的真实 `DOM` ，构建一个虚拟 `DOM` 树，

    ```js
    function createVNode(realDOM) { ... }
    ```

## createVNode 函数

- 例如

    ```js
    <div id="app">
      <p>{{ name }}</p>
      <p>{{ age }}</p>
    </div>

    const vNode = createVNode(document.getElementById("app"));
    ```

- 得到的 VNode 对象格式如下

    ```js
    // VNode 构造函数
    VNode: {
      realDom: div#app,
      template: "",
      children: [
        {
          VNode: {
            realDom: text,
            template: "    ",
            children:[],
          }
        }, {
          VNode: {
            realDom: p,
            template: "",
            children:[
              {
                VNode: {
                  realDom: text,
                  template: "{{name}}",
                  children:[],
                }
              }
            ],
          }
        }, {
          VNode: {
            realDom: text,
            template: "    ",
            children:[],
          }
        }, {
          VNode: {
            realDom: p,
            template: "",
            children:[
              {
                VNode: {
                  realDom: text,
                  template: "{{age}}",
                  children:[],
                }
              }
            ],
          }
        }, {
          VNode: {
            realDom: text,
            template: "    ",
            children:[],
          }
        }
      ]
    }
    ```
