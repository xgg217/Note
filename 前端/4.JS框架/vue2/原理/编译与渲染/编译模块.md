# 编译模块

## 概述

1. 将模板通过正则将变量取出

2. 将 `data` 定义的变量与 变量一一对应

## compile

+ `compile` 函数：将一个模板文本和环境对象编译成一个结果

    ```js
    function compile(template, envObj) { ... }
    ```

    ```js
    // 例如
    const resule = compile("姓名：{{ name }}，年龄：{{ age }}，居住省份：{{ addr.province }}省，{{ addr.city }}市",
    {
      name: "小刚刚",
      age: 16,
      addr: {
        province: "黑龙江",
        city: "哈尔滨"
      }
    });
    // 编译结果：姓名：小刚刚，年龄：16，居住省份：黑龙江省，哈尔滨市
    ```
