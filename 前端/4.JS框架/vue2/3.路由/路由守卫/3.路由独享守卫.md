# 路由独享守卫

## 作用

+ 是指在单个路由配置的时候也可以设置的钩子函数

## beforeEnter

+ 和 `beforeEach` 完全相同，如果都设置则在 `beforeEach` 之后紧随执行

    ```js
    const router = new VueRouter({
      routes: [
        {
          path: '/home',
          component: Home,
          beforeEnter: (to, from, next) => {
            // ...
          }
        }
      ]
    })
    ```
