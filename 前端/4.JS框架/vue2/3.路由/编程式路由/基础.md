# 基础

## 基础

  - 通过在 Vue 根实例的 `router` 配置传入 `router` 实例，`$router`、 `$route` 两个属性会被注入到每个子组件

## \$router

  - 路由实例对象

  - 除了使用 `<router-link>` 创建 `a` 标签来定义导航链接，我们还可以借助 `router` 的实例 方法，通过编写代码来实现

## \$route

  - 只读，路由信息对象

    ```javascript
    $route = {
      fullPath: "",
      hash: "",
      matched: [],
      meta: {},
      name: "",
      params: {},
      path: "",
      query: {}
    }
    ```
