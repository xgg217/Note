# 组件样式作用域

## scoped

*   html

    ```html
    <template>
      <button class="button button-close">X</button>
    </template>
    ```

*   css

    ```css
    <!-- 使用 `scoped` attribute -->
    <style scoped>
    .button {
      border: none;
      border-radius: 2px;
    }

    .button-close {
      background-color: red;
    }
    </style>
    ```

## CSS Modules

*   html

    ```html
    <template>
      <button :class="[$style.button, $style.buttonClose]">X</button>
    </template>
    ```

*   css

    ```css
    <!-- 使用 CSS Modules -->
    <style module>
    .button {
      border: none;
      border-radius: 2px;
    }

    .buttonClose {
      background-color: red;
    }
    </style>
    ```
