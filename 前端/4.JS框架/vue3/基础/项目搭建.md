# 项目搭建

## 方式1：命令行创建项目

*   `vue create xxx`

## 方式2：vite

*   npm

    ```bash
    $ npm init vite-app <project-name>
    $ cd <project-name>
    $ npm install
    $ npm run dev
    ```

*   yarn

    ```bash
    $ yarn create vite-app <project-name>
    $ cd <project-name>
    $ yarn
    $ yarn dev
    ```
