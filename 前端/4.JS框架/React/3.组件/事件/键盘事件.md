# 键盘事件

## 键盘事件监听

  - 代码

    ```react&#x20;jsx
    inputKeyUp = (event) => {
      console.log(event.keyCode); // 监听键盘
      if(event.keyCode === 13) {
        // 按 回车键触发
        const val = event.target.value;
        console.log(val); // 打印输入框的内容
      }
    }

    render() {
      return (
        <>
          <input onKeyUp={this.inputKeyUp} />
        </>
      );
    }
    ```
