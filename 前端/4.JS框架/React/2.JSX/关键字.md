# 关键字

## class 关键字，改成 className

- 错误

    ```html
    <div class="App"></div>
    ```

- 正确 `className`

    ```html
    <div className="App"></div>
    ```

## for 关键字，需要改成 htmlFor

- `htmlFor`

    ```html
    // html 标签
    <label for="inp">name</label>
    <input type="text" id="inp">
    ```

    ```react&#x20;jsx
    <label htmlFor="inp">name</label>
    <input type="text" id="inp">
    ```
