# 时间选择器

## 基本使用

  - 代码

    ```html
    <div class="layui-inline"> <!-- 注意：这一层元素并不是必须的 -->
      <input type="text" class="layui-input" id="test1">
    </div>
    ```

    ```js
    <script src="./layui/layui.js"></script>
    <script>
      layui.use(['laydate'], function() {

        var laydate = layui.laydate;

        //执行一个laydate实例
        laydate.render({
          elem: '#test1' //指定元素
        });
      })
    </script>
    ```
