# 生命周期

## 作用

*   即在特殊的时间点或遇到一些特殊的框架事件时，被自动触发的函数

    *   `created`：组件实例被创建好之后触发，组件在在调用 Component 里的定义数据 `data` 来实例化数据，此时不能使用 `setData`

    *   `attached`：组件初始化完毕，进入页面页面节点树时被触发，几乎所有的初始化工作都在这个函数执行

    *   `detached`：当退出页面时，页面中如果还有节点树，此函数被触发

    *   ready：组件在视图层布局完成后执行

    ```javascript
    Component({
      lifetimes: {
        attached: function() {
          // 在组件实例进入页面节点树时执行
        },
        detached: function() {
          // 在组件实例被从页面节点树移除时执行
        },
      },
    })
    ```
