# 拨打电话

## 使用

  - wxml

    ```js
    <text class='dhhm' bindtap='telFun' data-tel='{{ jbxxObj.telePhone }}'>{{ jbxxObj.telePhone || '暂无' }}</text>
    ```

  - js

    ```js
    // 拨打电话
    telFun(e){
      const tel = e.currentTarget.dataset.tel;
      console.log(tel);
      if(tel === '暂无') {
        return;
      }
      wx.makePhoneCall({
        phoneNumber: tel
      })
    }
    ```
