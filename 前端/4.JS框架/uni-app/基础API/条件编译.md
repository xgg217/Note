# 条件编译

## 语法

*   参考 `https://uniapp.dcloud.io/platform?id=preprocessor`

*   以 `#ifdef` 或 `#ifndef` 加 `%PLATFORM%` 开头，以 `#endif` 结尾

*   `#ifdef` ：if defined 仅在某平台存在

*   `#ifndef` ：if not defined 除了某平台均存在

*   `%PLATFORM%` ：平台名称

    ```javascript
    // 仅出现在 App 平台下的代码
    #ifdef APP-PLUS
    需条件编译的代码
    #endif
    ```

    ```javascript
    // 除了 H5 平台，其它平台均存在的代码
    #ifndef H5
    需条件编译的代码
    #endif
    ```

    ```javascript
    // 在 H5 平台或微信小程序平台存在的代码（这里只有||，不可能出现&&，因为没有交集）
    #ifdef H5 || MP-WEIXIN
    需条件编译的代码
    #endif
    ```
