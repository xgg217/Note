# 类型断言与类型转换

## 概述

+ 类型断言只会发生在 TS 的编译期，类型断言语句在编译结果中不存在

  ```ts
  fucntion toBool(som:any): boolean {
    return som as boolean
  }

  toBool(1) // 1

  // 虽然将 som 断言成 boolean 可以通过编译，但是并没有什么用，编译后的代码是
  fucntion toBool(som) {
    return som
  }
  toBool(1) // 1
  ```

+ 所以类型断言不是类型转换，它不会真的影响到变量的类型
+ 若要进行类型转换，需要直接调用类型转换的方法

  ```ts
  fucntion toBool(som:any): boolean {
    return Boolean(som)
  }

  toBool(1) // true
  ```
