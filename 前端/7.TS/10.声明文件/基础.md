## 概述

- 以 `.d.ts` 结尾的文件

## 声明文件的作用

- TS 代码读取 JS 代码，得不到类型声明

- 为JS代码提供类型声明

## 声明文件的位置

- 情况1：放置到 `tsconfig.json` 配置中包含的文件目录中

    ```js
    // src 文件下
    "include": ["src"]
    ```

- 情况2：放置到 `node_modules/@types` 文件夹中

- 情况3：手动配置：在 `tsconfig.json` 添加手动设置

    ```js
    "compilerOptions": {
      typeRoots": ["./types"]
    }
    "
    ```

- 情况4：与JS代码所在的目录，并且文件名也相同的文件。用 TS 代码书写的工程发布之后的格式

    ```js
    test.js
    ```
