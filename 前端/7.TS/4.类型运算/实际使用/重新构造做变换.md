## 索引类型

  - 索引类型可以添加修饰符 readonly（只读）、?（可选）

    ```ts
    type obj = {
      readonly name: string;
      age?: number;
      gender: boolean;
    }
    ```

    ```ts
    type Mapping<Obj extends object> = {
      [Key in keyof Obj]: Obj[Key]
    }
    ```
