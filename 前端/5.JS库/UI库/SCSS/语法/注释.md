# 注释

## 多行注释

*   多行注释会编译到`.css`文件中,`compressed`（压缩）模式下除外， 将 `!`作为多行注释的第一个字符表示在压缩输出模式下也保留这条注释，通常用于添加版权信息

    ```sass&#x20;(scss)&#x20;
    /*!
    * 我是
    * 多行
    * 注释
    */
    body { color: black; }
    ```

    ```css
    /* 编译成css */
    /*!
    * 我是
    * 多行
    * 注释 
    */body{color:#000}

    ```

## 单行注释

*   单行注释不会编译到`.css`文件

    ```sass&#x20;(sass)&#x20;
    // 我是单行注释
    body { color: black; }

    ```

    ```css
    /* 编译成css */
    body {
      color: black;
    }

    ```
