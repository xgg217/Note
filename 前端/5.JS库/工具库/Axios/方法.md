# 方法

## axios.request(config)

*   请求方式

    ```javascript
    axios.request({
      method: "GET",
      url: "http://localhost:3000/posts"
    }).then(res => {
      console.log(res.data);
    }).catch(err => {
      console.error(err);
    });
    ```

## axios.get(url,config)

*   请求方式

    ```javascript
    axios.get('http://localhost:3000/posts',{
    }).then(res => {
      console.log(res.data);
    }).catch(err => {
      console.error(err);
    });
    ```

## axios.post(url,config)

*   请求方式

    ```javascript
    axios.post("http://localhost:3000/posts", {
      data: {
        title: '修改2',
        auttor: '1233'
      }
    }).then(res => {
      console.log(res.data);
    }).catch(err => {
      console.error(err);
    });
    ```
