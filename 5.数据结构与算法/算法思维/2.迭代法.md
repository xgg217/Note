# 迭代法

## 概述

+ 英语叫做 Iteration

## 核心思想

+ 重复执行一系列步骤，直到满足某个条件或达到预定目标
+ 每次迭代都在一定程度上修改问题的状态，最终通过不断的更新和计算得到问题的解

## 核心步骤

+ 步骤1 初始化：先设置一个初始值，准备进入迭代

+ 步骤2 条件判断：在每次迭代开始之前，需要检查当前状态是否满足结束条件，如果满足结束条件，那么就停止迭代

+ 步骤3 更新：需要不停的去更新状态，从而进入下一次迭代

+ 步骤4 重复：一直执行迭代

## 示例之 循环

+ 其实，循环就是典型的迭代的思想

+ 举例：计算递增序列的和

  + 1+2+3+4+ ....+ 99+100

  ```js
  // 背后采用的就是迭代的思想
  let sum = 0;
  for(let i = 1; i <= 100; i++){
    sum += i;
  }
  ```
