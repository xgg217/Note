# 递归

## 概述

+ 递归是典型的分治的思想

+ 一种特殊形式的调用形式，指的是函数自己调用自己的形式

  ```js
  // 死递归
  function a(){
    a();
  }
  a();
  ```

## 递归的特点

1. 自调用，就是在函数里面调用自己
2. 最关键的一点，就是一个递归必须明确结束条件，不然就会陷入无穷的死循环
3. **缺点**，就是消耗大量内存

## 示例

+ 示例1 N 的阶层

  + `n! = n * (n -1)!;`
  + 所以：`5!` 的阶层等于 `5 * 4 * 3 * 2 * 1`

    ```js
    const f1 = (n) => {
      // return
      if(n == 1) {
        return 1;
      }
      return n * f1(n-1);
    }
    console.log(f1(3));
    ```


+ 示例2 第一年薪资是10k，涨幅每年5%，那么50年后薪资多少钱？

  ```js
  const nx = (n, t = 0.05) => {
    if(n === 1) {
      return 10000;
    }
    return nx(n - 1) * (1 + 0.05);
  };
  ```

  + 一共领取了多少钱

    ```js
    const allRMB = (n, t = 0.05) => {
      if(n === 1) {
        return 10000;
      }
      return nx(n, t) + allRMB(n - 1);
    }
    ```

+ 示例3 求一个数组前n项和

  ```js
  const arrSum = (arr = [], n = (arr.length)) => {
    const index = n - 1;
    if(index === 0) {
      return arr[0];
    }
    // debugger
    return arr[index] + arrSum(arr, index);
  }
  const arr = [1,2,3,4,5,6,7,8,9];
  console.log(arrSum(arr))
  ```
