# 递归

## 概述

+ 递归是典型的分治的思想

## 什么是递归

+ 一种特殊形式的调用形式，指的是函数自己调用自己的形式

  ```js
  // 伪代码
  function a(){
    a();
  }
  a();
  ```

## 注意点

+ 委托的思想：一个递归函数，告诉你是什么功能，你就不要想内部怎么实现的，反正委托给它，它就能给你正确的结果

+ 设计出口：什么时候是递归的终点，一旦没有设计出口，就会变成死递归

## 示例

+ n 的阶乘 n * (n - 1) * (n-2) * ... * 1

  ```
  5! = 5 * 4 * 3 * 2 * 1

  5! ---> 5 * 4!
  4! ---> 4 * 3!
  3! ---> 3 * 2!
  2! ---> 2 * 1!
  1! ---> 1(出口)
  ```

  ```js
  f(n); // 返回 n 的阶乘

  function f(n){
    // 出口非常重要，不会再继续往下调用了
    if(n === 1){
      return 1;
    }
    return n * f(n-1);
  }
  ```

+ 斐波那契数列

  ```
  0 1 1 2 3 5 8 13 21...

  // 求出指定位数的斐波那契数
  f(1) ---> 0
  f(5) ---> 3
  f(位数) ---> 对应位数的斐波那契数
  ```

  ```js
  function f(n){
    // 设计出口
    if(n === 1){
      return 0;
    } else if(n === 2){
      return 1;
    } else {
      return f(n-1) + f(n-2);
    }
  }
  ```
