# express基础

## 安装

+ `npm install express --save`

## 简单创建

+ 创建

  ```javascript
  const express = require("express");

  /**
  * 创建一个 express 应用
  * app 实际上是一个函数，用于处理请求的函数
  */
  const app = express();
  const port = 3000

  // 监听端口
  app.listen(port, () => console.log(`Example app listening on port ${port}!`))
  ```
