# API

## 下载

+ `npm i koa`

+ ts 还需要安装 `npm i --save-dev @types/koa`

## 创建 Koa 应用

+ 方式1

  ```javascript
  // ts 版本
  import Koa from 'koa';
  import http from 'http';
  const app = new Koa();
  const server = http.createServer(app.callback());

  server.listen(8080, () => {
    console.log('启动');
  });
  ```

  ```javascript
  // js版本
  const Koa = require('koa');
  const app = new Koa();
  const http = require('http');
  const server = http.createServer(app.callback());

  server.listen(port, callback);
  ```

+ 方式2：

  ```javascript
  const Koa = require('koa');
  const app = new Koa();

  app.listen(port, callback);
  ```

## 注册中间件

+ 注册中间件

  ```javascript
  app.use(middleware1);
  app.use(middleware2);
  app.use(middleware3);
  ```

+ 中间件格式

  ```javascript
  function(ctx, next) {
    // ctx 上下文对象
    // next() 移交给下一个中间件的函数
  }
  ```
