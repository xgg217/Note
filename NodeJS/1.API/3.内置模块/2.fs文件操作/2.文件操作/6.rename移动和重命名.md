# rename重命名

## 语法

+ `fsPromises.rename(oldPath, newPath)`

  + oldPath `<string>` | `<Buffer>` | `<URL>`
  + newPath `<string>` | `<Buffer>` | `<URL>`


  ```js
  // 重命名
  rename('./3.txt', './2.txt').then(res => {
    console.log(res);
  }).catch(err => {
    console.log(err);
  });
  ```

  ```js
  // 移动文件
  rename('./3.txt', './小/2.txt').then(res => {
    console.log(res);
  }).catch(err => {
    console.log(err);
  });
  ```
