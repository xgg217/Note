# 传递参数

## 传递参数

+ 传递单个参数

    ```javascript
    import { EventEmitter } from 'events';

    // 注册函数，同时获取参数
    const a1 = (data) => {
      console.log('adc 事件触发了1' + data);
    };

    // 注册事件
    ee.on('adc', a1);

    // 触发事件：同时传递参数
    ee.emit('avb', 123);
    ```

+ 传递多个参数

    ```javascript
    // 触发事件：同时传递参数
    ee.emit('avb', 123, 456);
    ```
